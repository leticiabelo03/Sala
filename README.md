# 📌 Sobre o Projeto

O **SALA** (Sistema Administrativo de Logística e Atendimento) é uma aplicação web corporativa desenvolvida para **centralizar operações administrativas, suporte técnico e gestão financeira** em uma única plataforma.

O projeto foi construído simulando um cenário empresarial real, onde diferentes setores trabalham conectados por fluxos automatizados.

*Professor:* Andre Muniz @agdelira

---

# 🎯 Objetivo

Criar um sistema interno capaz de integrar:

|       Área        |   Responsabilidade |
|-------------------|------------------|
|👑 Administrador   | Controle total e visão geral |
| 🛠️ Técnico        | Chamados e Ordens de Serviço |
| 💰 Financeiro     | Faturas e Pagamentos |
